
# Blackjack in C++

A terminal-based Blackjack game written in C++ featuring betting mechanics, AI dealer , and ASCII card graphics.

## Features

- Standard Blackjack gameplay
- Dealer AI
- Betting system
- Deck shuffling
- Multiple game rounds
